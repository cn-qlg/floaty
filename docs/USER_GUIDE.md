# Floaty 使用手册

适用版本：v0.1.1+ · macOS / Windows

---

## 快速上手 1 分钟

1. 安装后**首次启动**会自动弹出一张**上手指南便签** —— 直接读就行，教程内容按你的操作系统（macOS / Windows）自动适配
2. 上手指南便签可随时从 **菜单栏/系统托盘图标 → 📖 上手指南** 或 **偏好设置 → 打开上手指南** 重新打开

---

## 1. 创建便签

三种方式，任选：

- **菜单栏/系统托盘图标 → ＋ 新建便签**
- **快捷键 `⌘⇧N`（macOS）或 `Ctrl+Shift+N`（Win）** —— 全局，任何 app 下都能触发
- **便签聚焦时按 `⌘N` / `Ctrl+N`** —— 快速再加一张

---

## 2. 写内容

**完全不会 markdown？没关系**，两个浮动工具栏解决所有问题：

| 场景 | 工具栏 | 内容 |
|------|--------|------|
| 光标在空行 | 行尾自动弹出 | `☐ 待办` / `• 列表` / `1. 有序` / `H1` / `❝ 引用` |
| 选中文字 | 选区上方弹出 | 粗 / 斜 / 删除线 / 行内码 / H1 H2 / 🔗 链接 |

不想要工具栏？**偏好设置 → 编辑器** 分别关掉两个。

**会 markdown 的**可以直接打：

| 语法 | 效果 |
|------|------|
| `# 一级标题` | 大标题 |
| `## 二级` · `### 三级` | 小标题 |
| `- [ ] 待办` · `- [x] 已完成` | checkbox |
| `- 普通项` · `* 项` | 无序列表 |
| `1. 第一项` | 有序列表 |
| `> 引用内容` | 引用块 |
| `**粗**` / `*斜*` / `~~删~~` / `` `码` `` | 内联格式 |
| `---` | 水平线 |
| \`\`\`lang \n 代码 \n \`\`\` | 代码块 |
| `[Floaty](https://github.com/cn-qlg/floaty)` | 链接（⌘+点击打开） |

## 3. 加截止时间

**三种输入方式：**

### A. 按 `@` 打开选择器

快捷选项（今天 / 今晚 / 明天 / 周末 / 下周一）+ 倒计时（15 分钟 / 30 分钟 / 1 小时 / 2 小时 + 自定义）+ 具体日期时间（`<input type="datetime-local">`）。

### B. 直接用自然语言打字 + Tab

在文字尾部或中间任意位置打这些，行尾会出现**虚线 ghost 预览 + Tab 提示**，按 Tab 接受：

- **相对**：`30 分钟后` / `1 个小时后` / `3 天后` / `30m` / `2h`
- **今天**：`今天` (→ 今天 18:00) / `今晚` (22:00) / `今天早上` (08:00) / `今天下午` (15:00) / `今晚 19 点` (19:00)
- **明天/后天**：`明天` / `明天下午 3 点` / `明天 9:30` / `后天` / `大后天`
- **星期**：`周三` / `下周一` / `星期五` / `礼拜天`
- **绝对**：`4 月 25 日` / `4月25日` / `2026-05-15` / `18:30`
- **组合**：`后天下午 3 点` / `下周一早上 8 点` / `2026-05-15 14:30`
- **英文**：`tomorrow 9am` / `tomorrow 3pm`

所有模式容忍中英输入法的空格（`明 天 18 点` 也识别）。

### C. 点击已有 pill 改时间

点击便签里的彩色 pill → 再次弹出时间选择器（当前值会被预填）。

## 4. 紧急度自动染色

时间 pill 颜色根据距当前时间的距离自动变化，**每分钟刷新**：

| 条件 | 颜色 | 示例 |
|------|------|------|
| 已逾期 | 深红 `#c0392b` | `⚠ 逾期 2 小时` |
| 1 小时以内 | 红 `#e74c3c` | `⏰ 30 分钟` |
| 今天 | 橙 `#f39c12` | `⏰ 今天 18:00` |
| 本周 | 黄 `#f1c40f` | `⏰ 周五 10:00` |
| 更晚 | 灰 `#95a5a6` | `⏰ 5/1 10:00` |

## 5. 到期提醒

- 有 `@due` 的 todo 到时间会自动弹 **macOS/Windows 系统通知**
- 后台 Tokio 定时器调度，app 运行就有效
- 启动时会补发过去 1 小时内错过的提醒
- Snooze / 完成 / 查看等 UI 在后续版本加

---

## 6. 多便签管理

- **拖顶部栏**移动窗口
- **右下角拖拽** resize
- 顶部按钮：
  - `📌` 置顶（高亮红 = 置顶中）
  - `⚙` 外观设置（背景色 / 透明度 / 字号 / 字色 / 重置 / 删除）
  - `✕` 隐藏（DB 保留，从菜单栏可恢复）
- **菜单栏/系统托盘** → **显示全部** 把所有便签拉到最前
- **菜单栏/系统托盘** → **一键排版** 把所有可见便签按 4 列网格重新摆放（统一大小 320×420）

---

## 7. 外观自定义（每便签独立）

点 `⚙` 打开外观 popover：

- **背景色**：8 个预设 + 自定义 HEX
- **透明度**：30%–100% 滑块
- **字号**：小 12 / 中 14 / 大 16
- **字体色**：自动跟随背景明暗 / 自定义 HEX

## 9. 偏好设置

**菜单栏 → ⚙️ 偏好设置**：

- **统计**：便签 / Todo / 待触发提醒 数量
- **数据目录 & 备份**：
  - macOS: `~/Library/Application Support/app.floaty.desktop/floaty.db`
  - Windows: `%APPDATA%\app.floaty.desktop\floaty.db`
  - **立即备份到…**：一键把 DB 导出到任意位置（VACUUM INTO，运行时安全）
  - **自动快照**：每次 app 启动自动把 DB 复制到数据目录下的 `backups/`，保留最近 10 份（无感）
- **关于 & 更新**：当前版本 + 一键检查更新（自动下载安装并重启）
- **启动**：开机自动启动（macOS LaunchAgent / Windows 注册表）
- **编辑器**：分别开关两个浮动工具栏
- **快捷键**：全局开关 + 当前所有快捷键列表
- **操作**：新建便签 / 打开上手指南

## 常见问题

### 便签怎么找不到了？
1. **菜单栏 → 一键排版** 把所有便签摆到屏幕左上角
2. 或 **菜单栏 → 显示全部** 把隐藏的全部恢复并置顶

### 关了便签 app 就退出了？
不会。关掉所有便签后 app 仍在菜单栏常驻。要彻底退出：**菜单栏 → 退出**。

### 已经删掉了能恢复吗？
不能。`✕` 只隐藏（可恢复），`⌘⌫` 或**设置 → 删除便签** 才是永久删除（含 items + reminders）。

### 重装或换电脑数据迁移？
复制整个数据目录到新机器同位置即可。DB 是 SQLite 单文件。

### 数据出问题了，怎么从备份恢复？
数据目录里有两处备份来源：
- `backups/floaty-YYYYMMDD-HHMMSS.db`：app 每次启动自动生成，保留最近 10 份
- 你用"立即备份到…"手动导出的那份

恢复步骤：
1. **菜单栏 → 退出**（务必退干净，否则文件被占用）
2. 打开数据目录（偏好设置里"在 Finder / 资源管理器中显示"）
3. 把当前的 `floaty.db` 重命名成 `floaty.db.bad`（先不删，万一）
4. 把要恢复的备份文件复制一份进来，改名为 `floaty.db`
5. 重新启动 app

如果连启动自动快照都丢了（例如数据目录被整体清掉），就只剩你手动备份那份了 —— 所以重要阶段建议手动导一份到网盘 / Time Machine / U 盘等外部位置。

### 时间识别不准？
在时间表达后按 Tab 接受 ghost pill 时，pill 里会显示解析的**绝对时间戳**（`2026-04-21 18:00:00`）。不对就按 Esc 取消重打。

### 怎么卸载？
- macOS: 删 `/Applications/Floaty.app` + 删数据目录
- Windows: 控制面板卸载 + 删 `%APPDATA%\app.floaty.desktop\`
